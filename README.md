- Smart Parking Management System using Discrete Structures Concepts :-
  
Smart Parking Management System in C++ implementing Discrete Structures concepts including sets, mappings, priority queues, and event-driven parking allocation.
Bridging Discrete Structures with a real-world parking solution through intelligent slot allocation, queue management, and event tracking.
An interactive console-based parking management system developed in C++ that applies real-world Discrete Structures concepts such as sets, mappings, priority queues, relations, and event logging. The system dynamically allocates parking slots, manages vehicle entries and exits, handles waiting queues, tracks parking utilization, and maintains a complete activity log for efficient parking operations.
