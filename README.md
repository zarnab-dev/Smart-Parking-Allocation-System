# Smart Parking Allocation System

A console-based Smart Parking Management System developed in **C++** that applies key **Discrete Structures concepts** to solve a real-world parking allocation problem.

## Project Overview

This system allows users to create parking slots, register vehicles, allocate the most suitable parking space, manage waiting queues, process vehicle exits, and generate complete parking activity logs.

The project demonstrates the practical implementation of Discrete Structures concepts such as:

* Sets
* Relations
* Functions (Vehicle → Parking Slot Mapping)
* Priority Queues
* Event Logging
* Data Organization and Allocation

## Features

* Dynamic parking slot creation
* Vehicle registration and allocation
* Intelligent slot assignment
* Waiting queue management
* Vehicle exit processing
* Parking utilization tracking
* Complete event logging system
* Interactive user-based console interface

## Discrete Structures Concepts Used

| Concept               | Application                |
| --------------------- | -------------------------- |
| Sets                  | Managing parking slots     |
| Relations             | Vehicle-slot associations  |
| Functions             | Allocation mapping a(v)    |
| Priority Queues       | Waiting vehicle management |
| Graph/Event Tracking  | Activity logging           |
| Mathematical Modeling | Parking allocation process |

## System Workflow

1. User creates parking slots.
2. User enters vehicle information.
3. System allocates the best available slot.
4. Full slots trigger waiting queue management.
5. Vehicle exits free occupied slots.
6. Waiting vehicles are automatically allocated when slots become available.
7. Event logs record all parking activities.

## Output

The system displays:

* Slot allocation results
* Parking utilization statistics
* Available and occupied slots
* Vehicle exit records
* Complete event logs
